# Jewellery CRM (Electron + Express + Pug + Sequelize)

This is a starter scaffold for an offline desktop CRM tailored for a jewellery manufacturing business. It uses Electron for the desktop shell, Express + Pug for the UI, and Sequelize with SQLite for an offline database.

Getting started (Windows / PowerShell)

1. Install dependencies

```powershell
npm install
```

2. Start the app (Electron)

```powershell
npm run start
```
